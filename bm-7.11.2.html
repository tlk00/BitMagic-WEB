Version 7.11.2

Feb 14, 2022

Release Notes

  1. Fixed bug in rank-select index assisted compute of rank (affecting compressive sparse vector) bm::bvector<>::count_to(...)

  2. Fixed bug in deserialization of empty sparse vectors

  3. C language interface (libbm) – fixed incorrect handling of CPU SIMD capabilities (AVX2)

  4. Optimization: Improved read-only mode for bm::bvector<> to allocate one contiguos memory block for the arena to better reduce memory fragmentation. This improvement also affects read-only mode memory allocation for bit-vector based succinct vectors.

  5. Optimization: succinct vector for strings (bm::str_sparse_vector) remap() method improved to use less memory

  6. Added bit-vector immutability methods into C-language mappings.
